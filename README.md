# ChatGPT-Extension
 A google chrome extension that allows you to use chatGPT from any webpage by clicking the extension icon. 

# Install
1. Download the zip of the code from Github
2. Locate the Chrome extensions page and turn on "Developer Mode"
3. Click on "Load Unpacked" and select on myChatGPT
4. Enjoy!

# Note
You must be logged into chatGPT in order for this extension to work.
